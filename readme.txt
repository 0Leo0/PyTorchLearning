jupyter notebook常用的快捷键操作：

ESC：退出编辑

    A：在上面插入一行

    B：在下面插入一行

    M：转为Markdown模式

    Y：转为code模式

Enter：开始编辑

    Shift+Enter：运行cell并在下面插入一行

    Ctrl+Enter：仅仅运行cell
